componentes basicos do reactnative

view:
    - é o container principal do reactnative
    - é o elemento mais importante do reactnative
    - é o elemento que contem todos os outros elementos

Image:
    - é um componente que mostra uma imagem
    - pode ser usado para mostrar imagens de fundo
    - pode ser usado para mostrar imagens de icones
    - é um elemento que representa uma imagem
    - é um componente
    - é um elemento do reactnative

text:

textinput
